How to use a Div:

A div goes between a chunck of code you want to edit, you can
call upon this div by first setting a class to the div, for example
< div class="Page2Paragraph1-3"> info goes here < /div> - then close the div.
in order to call upon that class in the div, you do .Page2Paragraph1{ attributes }

How to use a Span:

A span goes inside an element like < p> and you put it in between the element
like this: < p < span class="highlight"> info < /span>
So anything inbetween the two spans will be highlighted. Just call
upon the class using ."class name" {attributes}

How to use an id:

id's work a lot like a class, you attach an id to an element like a div.
makes an individual set of code that only applies to one element. Used for YOU
to be able to read your code better. Ex. < div id="divPara1"> info < /div>
You can call upon this using the #divPara1 {attributes}

How to use a float:

The float is used for setting a specific area or paragraph to the side of
wall for example by adding the float: right; to a paragraph class, it will
make everything in that paragraph sick two the right wall and this is how
you place things side by side without using the position command. This
also works with the width command, put width to 50% and the two paragraph
split the screen.
